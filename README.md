Homebrew-arduino
================
Brew formulae to be able to use inotool for commandline Arduino programming.

Installing Homebrew-arduino formulae
------------------------------------
You can tap this repository using:

```brew tap wardwouts/arduino```

Then install inotool using:
```brew install ino```
